# Heart Firework PWA (GitHub Pages)

## 1) 檔案
- index.html
- manifest.json
- service-worker.js
- icon.svg
- icon-192.png
- icon-512.png

## 2) GitHub Pages
Repository Settings → Pages
- Source: Deploy from a branch
- Branch: main / (root)
儲存後等待 Pages 生成網址。

## 3) iPhone 安裝（像 App）
1. 用 Safari 打開 Pages 網址（HTTPS）
2. 進入後會看到「Tap to enable motion」提示，點一下允許陀螺儀
3. Safari 分享 → 加到主畫面
4. 從桌面圖示開啟（standalone）

## 4) icon-192 / icon-512
把 icon.svg 轉成 PNG：
- icon-192.png (192x192)
- icon-512.png (512x512)
